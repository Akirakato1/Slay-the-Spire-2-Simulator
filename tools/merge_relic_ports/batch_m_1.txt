// const declarations and execute_X_move bodies.
//
// Conventions:
//   - Intent string key matches the PascalCase enum variant name
//     (e.g. "BootUp" for AxebotIntent::BootUp). Mirrors the existing
//     Axebot entries in monster_move_effects.
//   - SelfActor → moving monster, ChosenEnemy → player (single-player).
//   - Pure-display intents (SleepIntent / animation-only) encode as
//     `Some(vec![])` so the dispatcher resolves them through the VM
//     without firing payload effects. Match the Rust no-op semantics.
//   - Axebot intentionally not re-emitted (already in monster_move_effects).
//   - MysteriousKnight aliases FlailKnight payloads (monster_dispatch
//     routes them through pick/execute_flail_knight_move). Emit both
//     monster_ids so the registry can match either.
