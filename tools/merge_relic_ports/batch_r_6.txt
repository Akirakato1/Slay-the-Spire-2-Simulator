// batch_r_6.txt — bulk-encoded combat-frame relic hooks for the remaining ~190
// relic .cs files in sts2_decompiled/.../Core/Models/Relics/ that weren't
// covered by batch_r_1..5. Format mirrors batch_r_1 (RelicHook + Effect list).
//
// Strategy:
//   * AfterRoomEntered(CombatRoom) bodies map to BeforeCombatStart.
//   * ModifyMaxEnergy(+N) maps to BeforeCombatStart + IncreaseMaxEnergy(N).
//   * ModifyHandDraw round-1-only maps to BeforeCombatStart + DrawCards.
//     Per-turn ModifyHandDraw maps to AfterPlayerTurnStart + DrawCards
//     gated on Round/condition where possible.
//   * AfterPlayerTurnStartLate round<=1 → AfterPlayerTurnStart first_turn_only.
//   * BeforeHandDraw round 1 → BeforeCombatStart (initial draw fires after).
//   * AfterRoomEntered RestSite / etc. that gates on out-of-combat is SKIPped
//     (those are run-state hooks; we only get combat-frame hooks here).
//   * AfterAttack, AfterShuffle, AfterCardChangedPiles, AfterDeath, AfterEnergyReset,
//     AfterCreatureAddedToCombat, AfterStarsSpent, AfterDiedToDoom,
//     AfterPotionUsed, AfterPotionProcured / Discarded, AfterModifyingRewards,
//     AfterHandEmptied, AfterCardEnteredCombat,
//     AfterPreventingDeath, AfterPreventingBlockClear, AfterRestSiteHeal,
//     AfterModifyingCardPlayCount, AfterModifyingOrbPassiveTriggerCount,
//     AfterModifyingHandDraw, BeforeCardPlayed, BeforePowerAmountChanged,
//     ModifyDamage*, ModifyBlock*, ModifyPowerAmount*, ModifyRestSiteHealAmount,
//     ModifyMerchantPrice, ModifyHpLost*, TryModifyRewards*, TryModifyCard*,
//     ShouldGainGold, ShouldProcurePotion, BeforePlayPhaseStart*
//     are SKIPped (hook surfaces not wired).
//   * Already-in-dispatcher relics (Anchor, BurningBlood, Brimstone) are NOT
//     re-emitted (kept as hand-coded match arms in combat.rs).

// BurningSticks.cs L46-58: AfterCardExhausted, owner skill, once per combat →
// Adds a copy of the exhausted skill to hand. Counter starts at 1 at combat
// start; firing the clone+decrement guards against repeat fires.
"BurningSticks" => Some(vec![
    (RelicHook::BeforeCombatStart,
     vec![Effect::SetRelicCounter { key: "BurningSticks_charges".to_string(), value: AmountSpec::Fixed(1) }]),
    (RelicHook::AfterCardExhausted,
     vec![Effect::Conditional {
        condition: Condition::And(
            Box::new(Condition::SourceCardHasKeyword("Skill".to_string())),
            Box::new(Condition::RelicCounterGe { key: "BurningSticks_charges".to_string(), value: 1 }),
        ),
        then_branch: vec![
            Effect::CloneSourceCardToPile {
                pile: Pile::Hand,
                cost_override_this_combat: None,
                copies: AmountSpec::Fixed(1),
            },
            Effect::ModifyRelicCounter { key: "BurningSticks_charges".to_string(), delta: AmountSpec::Fixed(-1) },
        ],
        else_branch: vec![],
     }]),
]),

// ===== Summary =====
// New combat-frame: 28
// New run-state: 0 (run-state entries live in batch_r_rs_3)
// Still SKIPped: ~80 combat-only relics
//   Top SKIP blocker categories:
//     1. Modify* / TryModify* damage/block/power/cost/reward hooks (~30) — would need
//        modifier-hook surface (damage-pipeline, block-pipeline, reward-pipeline,
//        cost-pipeline).
//     2. AfterCardChangedPiles / AfterShuffle / AfterCardEnteredCombat / AfterAttack
//        / AfterDeath / AfterDiedToDoom / AfterHandEmptied / AfterPotionUsed
//        / AfterOrbChanneled / AfterEnergyReset / AfterStarsSpent / AfterRestSiteHeal
//        / AfterPreventing* / AfterModifying* / AfterCreatureAddedToCombat (~22)
//        — niche hook surfaces not currently wired.
//     3. BeforeCardPlayed / BeforePlayPhaseStart / BeforeHandDraw / BeforePowerAmountChanged
//        / BeforeTurnEndEarly (~12) — pre-action hook surfaces not surfaced.
//     4. Interactive run-state mutation (transform/enchant/select-from-deck/
//        reward-offer) — these have no combat-frame side; covered in batch_r_rs_3.
//     5. RoomType-filtered AfterRoomEntered (Elite / RestSite / Merchant /
//        Unknown / BaseRoom) (~9) — needs run-state hook with room_type_filter.
//     6. Empty / cosmetic relics (~10) — no override body to encode.
