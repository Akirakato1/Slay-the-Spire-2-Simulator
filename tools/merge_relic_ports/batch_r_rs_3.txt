// are now unblocked by AddCardToRunStateDeck (mid-run deck additions) and
// LoseRunStateMaxHp.
//
// Allowed run-state Effect variants:
//   GainRunStateMaxHp / GainRunStateGold / LoseRunStateHp / LoseRunStateMaxHp /
//   GainRelic / LoseRelic / GainPotionToBelt / AddCardToRunStateDeck /
//   GainMaxPotionSlots
//
// AmountSpec::Canonical resolves to 0 at run-state scope, so literal Fixed(N)
// is used where N is the C# DynamicVar BaseValue.
//
// SKIPs document the precise reason — usually because the C# body relies on
// a primitive the dispatcher doesn't yet implement.
